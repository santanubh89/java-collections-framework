Stack is a LIFO data structure, meaning the last element added to the stack is the first one to be removed. It operates like a collection of items where you can only add or remove items from the top of the stack.
Has methods:
push(E item): Adds an item to the top of the stack.
pop(): Removes and returns the item from the top of the stack.
peek(): Returns the item at the top of the stack without removing it.
isEmpty(): Checks if the stack is empty.
size(): Returns the number of items in the stack.

Contains methods similar to LinkedList like addFirst(), removeFirst(), getFirst() etc.
